## Quick Setup Guide

Get up and running with Onuro's AI-enhanced development tools quickly.

### Choose Your Platform

<AccordionGroup>
  <Accordion icon="globe" title="Web & Mobile Apps">
    1. Visit [app.onuro.ai](https://app.onuro.ai)
    2. Sign in or create an account
    3. Start using Onuro's AI assistant right in your browser
  </Accordion>

  <Accordion icon="code" title="IDE Extensions">
    1. Install the Onuro extension for your IDE:
       - [VS Code Extension](https://marketplace.visualstudio.com/items?itemName=onuro.vscode)
       - [JetBrains Extension](https://plugins.jetbrains.com/plugin/onuro)
    2. Sign in with your Onuro account
    3. Start using AI-powered development tools
  </Accordion>

  <Accordion icon="chrome" title="Chrome Extension">
    1. Install Onuro from the [Chrome Web Store](https://chrome.google.com/webstore/detail/onuro)
    2. Sign in with your Onuro account
    3. Access AI assistance anywhere while browsing
  </Accordion>
</AccordionGroup>

### First Steps

<Steps>
  1. Choose Your Environment
     - Web & Mobile Apps for general AI assistance
     - IDE Extensions for coding and development
     - Chrome Extension for browsing assistance

  2. Enable AI Features
     - Configure your preferred AI model
     - Set up context management
     - Customize your experience

  3. Start Using Onuro
     - Try code completion
     - Ask for explanations
     - Get AI-powered suggestions
</Steps>
